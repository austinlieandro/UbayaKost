# UbayaKost
An application that can display a list of boarding houses located near UBAYA.

## Features
- Display kost near UBAYA
- Show detail kost
- Review kost
- Save kost
- My kost
- History kost
- History transaction
- Using custom JSON
- Using navbar and bottombar
